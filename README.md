# Gym Management System (Database Project) 

A comprehensive relational database system designed to streamline and manage the operations of a professional fitness center or gym. This project focuses on database design, normalization, and efficient SQL querying.

##  Key Features
- **Member Management:** Tracks member details, registration dates, and active memberships.
- **Trainer & Staff Scheduling:** Assigns trainers to specific members and classes.
- **Membership & Subscription Plans:** Manages different tiers of packages and payments.
- **Payment & Invoicing:** Keeps history of membership renewals and transaction statuses.

##  Database Technologies
- **Language:** SQL
- **Database Management System (DBMS):** SQL Server / MySQL 
- **Concepts Applied:** Entity-Relationship (ER) Diagram, Normalization (1NF, 2NF, 3NF), Relational Schema, Constraints, and Foreign Key relations.

##  Database Schema / Structure
This database consists of several core tables to handle complex relationships:
- `Members`: Personal info and membership status.
- `Trainers`: Trainer details, specialties, and ratings.
- `Plans`: Pricing tiers and benefits.
- `Payments`: Tracking transaction IDs, dates, and amounts.
